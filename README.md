# Kaanilai Weather Application

A modern weather application built with jQuery that provides real time weather information through a clean and responsive user interface.

## About

Kaanilai is a weather forecasting application that allows users to quickly check the current weather conditions of any location.

The application fetches live weather data and presents it in an easy to understand format with dynamic weather visuals, making it simple for users to stay updated with changing weather conditions.

The goal is simple:

- Check real time weather
- Expect current day weather details
- Enjoy a clean and responsive weather experience

## Features

### Current Weather

View real time weather details including:

- Temperature
- Weather condition
- Wind Speed
- Humidity
- Sunrise and Sunset
- UV Index

### Dynamic Weather Backgrounds

Weather backgrounds automatically change based on the current weather condition for a more immersive experience.

### Responsive Design

Optimized for desktop, tablet, and mobile devices.

## Technologies Used

- HTML5
- Tailwindcss
- jQuery
- Open Meteo Weather API
- Ionic Icons

## Why This Project?

This project was created to practice building a real-world application using jQuery while working with external APIs.

It demonstrates:

- API integration
- AJAX requests
- Dynamic DOM manipulation
- Responsive UI design
- Weather based UI updates

## Getting Started

Clone the repository:

```bash
git clone https://github.com/awizp/kaanilai-weather-application.git
```

Navigate to the project folder:

```bash
cd kaanilai-weather-application
```

Open `index.html` in your browser or run it using your preferred local server.
